# tech_hub_api
Este projeto consiste em uma api de armazenamento de registro de compra de produtos e vendas.

## Rodar localmente
#### 1. Clone o projeto
    git clone https://github.com/RafaelFernandes12/tech_hub_API.git
#### 2. Abra a pasta do projeto
    cd tech-hub-api/
#### 3. Rode o projeto
    npm run dev

## Ferramentas
* Fastify js
* Node js
* Prisma
* Mysql
* Typescript
